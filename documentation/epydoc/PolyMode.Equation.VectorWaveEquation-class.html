VWE Magnetic equation class: diff: Difference class for finite differences Calculate the action of the VWE on a vector, defined as L^p H^p + G^{p,q=1} H^+ + G^{p,q=-1} H^- for x^p

Note: x must be shaped correctly, as x[Np, Nr, bw, Naz, Naz] with Np=2
