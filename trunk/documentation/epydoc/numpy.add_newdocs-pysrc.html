 439   
 440      A highly efficient way of reading binary data with a known data-type,

 442      `tofile` method can be read using this function. 
 443

 445      ----------

 458          Spaces (" ") in the separator match zero or more whitespace characters. 
 459          A separator consisting only of spaces must match at least one 
 460          whitespace. 
 461   
 462      See also

 467

 674   
 675      Return evenly spaced values within a given interval. 
 676

 678      (in other words, the interval including `start` but excluding `stop`). 
 679      For integer arguments the function is equivalent to the Python built-in

 692          between two adjacent values, ``out[i+1] - out[i]``.  The default 
 693          step size is 1.  If `step` is specified, `start` must also be given. 
 694      dtype : dtype 
 695          The type of the output array.  If `dtype` is not given, infer the data

 705          this rule may result in the last element of `out` being greater 
 706          than `stop`. 
 707

 711      ogrid: Arrays of evenly spaced numbers in N-dimensions 
 712      mgrid: Grid-shaped arrays of evenly spaced numbers in N-dimensions
