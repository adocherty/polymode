Class ImportBitmap

source code

Import a bitmap image (png) file

Parameters:

  • filename: name of image file (not file object)
  • density: dpi of image either single numer or tuple of (dpi_x, dpi_y)
  • size: size in um of image either single numer or tuple of (size_x, size_y) this is used to calculate the density, so only one of density, size should be given
  • hist: histogram function to use to normalize the image data
  • negative if False black is interpreted as no material, white is interpreted as presense of material if True the opposite interpretation is used
